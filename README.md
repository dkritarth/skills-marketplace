# Personal Skills Marketplace

Marketplace of highly fine-tuned skills, generated by me.

## Planned skills

- **LATEX** — highly fine-tuned LaTeX skill (beyond generic LaTeX help).
- **WRITING** — paper-writing skill that tracks modification history and
  translates it into future intent of the work (i.e. remembers where a
  draft is headed, not just its current state).
- **MATH** — proper structure for generating proofs and mathematical
  conceptual understanding.
- More TBD — needs further thought.

## Status

Scoped and finalized. Distribution model: a personal Claude Code plugin
marketplace, following the existing `~/.claude/plugins/marketplaces/`
pattern (like `caveman`), not standalone skill repos. WRITING is the
flagship/differentiator — prior-art research found no existing tool that
infers a paper draft's forward trajectory from revision history, via a
per-draft journal file backed by git. LATEX and MATH are built fresh but
scoped narrowly (MATH ships only if it adds real SymPy/Lean verification,
otherwise folds into LATEX), since both face real existing competition.

See `PLAN.md` for the full concept: problem statement, decisions, skill
specs, phased milestones, success criteria, and risks. See `AGENTS.md` and
`CLAUDE.md` for how to build and test skills in this repo.
